# 二叉树

子节点数最多为2的有序树。

![P449057594017](../../../pictures/P449057594017.png)

二叉树有如下特性：

- 第i层最多有$2^{i-1}$个节点；
- 如果深度为h，那么最多有$2^h$ - 1个节点。



## 满二叉树

叶子节点均在最后一层，且非叶子节点度都为2。

![P543375232913](../../../pictures/P543375232913.png)



## 完全二叉树

形如满二叉树叶子节点从右往左依次移除n（n >= 0）个节点的树。

![P947993736201](../../../pictures/P947993736201.png)

满二叉树一定是完全二叉树，完全二叉树不一定是满二叉树。

完全二叉树包含二叉树的所有特性，将完全二叉树从上到下从左到右排序，还有如下特性：

- 当前节点为i，则父节点为⌊i / 2⌋；
- 当前节点为i，则左孩子节点为$2^i$，右孩子节点为$2^i$ - 1。
