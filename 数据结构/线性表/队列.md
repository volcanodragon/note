# 队列

遵循先进先出规则的线性表，即只能从表的一端插入元素，另一端取出元素。取出的端被称为队头，插入的端被称为队尾。

![P509240132779](../../pictures/P509240132779.png)



## 顺序队列

即顺序存储的队列。通过队列的特性不难看出，在使用队列时，队头空闲单元会越来越多，队尾空闲单元会越来越少，且队头的空闲单元无法复用，即队列中的存储单元都是“一次性”的，极大的浪费了存储单元。一般来说不会采用顺序队列实现。



## 循环队列

将顺序队列的队头和队尾“连接”起来形成逻辑上的循环。循环队列解决了顺序队列存储单元浪费的问题，是队列的常用实现方式。

![P490321979304](../../pictures/P490321979304.png)

当队列为空时，front（队头指针） = rear（队尾指针）。当插入元素时，元素插入rear所指单元，然后rear后移；当取出元素时，元素从front所指单元取出，然后front后移。指针位移后值的计算方式为：front/rear = (front/rear + 1) % size。

按上述步骤可以推断出当队列填满时有front = rear，与队列为空的判断重合。一种常用的解决方式是在队尾空出一个存储单元，即队列最多只能存储size - 1个元素。这样满队列判断就为：front = (rear + 1) % size，与队列为空判断区分开来。
